Description

This object provides an interface for the CLASS code.

If you use this object please cite: Blas (2011) CLASS II, see also:

  • Lesgourgues (2011) CLASS I,

  • Lesgourgues (2011) CLASS III,

  • Lesgourgues (2011) CLASS IV and

  • CLASS website.
