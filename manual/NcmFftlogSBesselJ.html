Description

This object computes the function (see NcmFftlog) $$Y_n = \int_0^\infty t^{\frac{2\pi i n}{L}} K(t) dt,$$ where the kernel are the spherical bessel function of the first kind $K(t) = t^q j_{\ell}(t)$.
